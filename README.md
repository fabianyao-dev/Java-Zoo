# 🐾 Simulador de Zoológico en Java
Simulador de zoológico en Java demostrando dominio en Programación Orientada a Objetos (Herencia, Interfaces, Polimorfismo). Proyecto final de Coursera.

## 📖 Descripción general
Este proyecto es una aplicación desarrollada en Java que simula la gestión y el comportamiento de animales en un zoológico. Fue desarrollado como el proyecto final calificable de mi curso en Coursera, diseñado para poner en práctica la lógica de programación y el diseño de software estructurado.

El programa modela diferentes tipos de animales partiendo de una clase base `Animal`, la cual se extiende a clases concretas como `Tigre` (animal terrestre) y `Delfin` (animal acuático). El enfoque principal del proyecto es la **escalabilidad** y la **reutilización de código** a través de un diseño lógico robusto.

## ✨ Conceptos Clave de POO Implementados
Este repositorio demuestra el dominio práctico de los siguientes conceptos:

* **Herencia y Polimorfismo:** Creación de una clase base `Animal` que comparte propiedades comunes a sus subclases, permitiendo tratar distintos objetos de forma unificada.
* **Interfaces Selectivas:** Diseño de interfaces basadas en comportamientos del mundo real (ej. habilidades de nadar o caminar), aplicadas únicamente donde tiene sentido lógico (un delfín no implementa la acción de caminar).
* **Clases Abstractas:** Uso de abstracción para definir plantillas estrictas que las subclases deben seguir.
* **Encapsulamiento y Modificadores de Acceso:** Protección de la integridad de los datos dentro de las clases para un código más seguro y mantenible.
* **Diseño Modular:** Separación de responsabilidades utilizando clases de utilidad y una clase `Main` limpia para la ejecución.
